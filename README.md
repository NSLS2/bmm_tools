# bmm-profile-common
Common code for Bluesky profiles for BMM end stations and other tools

This repo contains code used in othe BMM repos:

+ The profile for XAS data collection: https://github.com/NSLS2/bmm-profile-collection

+ The profile for data collection using the goniometer (work in progress):
  https://github.com/NSLS2/bmm-profile-goniometer
  
+ The repo containing the kafka-based plot and file managers (still a
  part of the XAS profile, separating this code to its own repo is
  future work)

+ The cadashboard tool (future work)
